# Функции игры Mining Game

## Основная механика

### Платформа для добычи
- Платформа размером 10x10 клеток
- Изначально доступна только 1 клетка
- Клетки можно покупать за монеты
- Каждая клетка имеет размер 4x4 единицы

### Система камней
- Камни спавнятся автоматически каждые 30 секунд
- Разные типы камней с разной редкостью:
  - **Обычный камень** (60% шанс) - серый цвет
  - **Медь** (25% шанс) - коричневый цвет
  - **Олово** (10% шанс) - серебристый цвет
  - **Железо** (5% шанс) - темно-коричневый цвет

### Добыча ресурсов
- Клик левой кнопкой мыши по камню для добычи
- Анимация исчезновения камня
- Автоматическое добавление ресурсов в инвентарь
- Визуальные эффекты при добыче

## Экономическая система

### Цены ресурсов
- **Камень**: 1 монета
- **Медь**: 3 монеты
- **Олово**: 5 монет
- **Железо**: 10 монет

### Стоимость клеток
- Увеличивающаяся стоимость: 10, 20, 30, 40, 50... монет
- Формула: `количество_разблокированных_клеток * 10`

### Торговец
- Золотой NPC рядом с платформой
- Клик по торговцу продает все ресурсы
- Кнопка "Sell All" в интерфейсе

## Интерфейс

### Основной GUI
- Правый верхний угол экрана
- Отображение количества монет
- Инвентарь с количеством каждого ресурса
- Кнопки для покупки клеток и продажи ресурсов

### Управление камерой
- **Правая кнопка мыши** + движение - вращение камеры
- **Колесико мыши** - зум камеры
- Камера автоматически следит за центром платформы

## Визуальные эффекты

### Эффекты камней
- Анимация парения для всех камней
- Свечение для редких камней (медь, олово, железо)
- Разные цвета свечения для каждого типа

### Эффекты добычи
- Частицы дыма при добыче камня
- Текстовые эффекты с количеством полученных ресурсов
- Анимация исчезновения камня

### Эффекты разблокировки
- Зеленое свечение при покупке новой клетки
- Анимация расширения эффекта

## Технические особенности

### Серверная часть
- **GameManager**: Основная логика игры
- **RemoteHandler**: Обработка клиентских запросов
- **PlayerSpawner**: Размещение игроков
- **RemoteEventsSetup**: Настройка сетевого взаимодействия

### Клиентская часть
- **PlayerController**: Управление интерфейсом и взаимодействиями
- **CameraController**: Управление камерой
- **VisualEffects**: Визуальные эффекты

### Сетевое взаимодействие
- RemoteEvents для связи клиент-сервер
- Автоматическая синхронизация данных
- Обновление интерфейса в реальном времени

## Геймплей

### Начало игры
1. Игрок появляется над первой клеткой
2. Камера автоматически настраивается
3. Зеленая подсказка показывает доступную клетку

### Прогресс
1. Добыча камней на доступных клетках
2. Продажа ресурсов торговцу
3. Покупка новых клеток
4. Расширение территории добычи

### Цель
- Максимальное расширение территории
- Накопление ресурсов
- Эффективная торговля

## Настройки игры

### Конфигурация
- Размер сетки: 10x10
- Размер клетки: 4 единицы
- Интервал спавна камней: 30 секунд
- Начальное количество клеток: 1

### Шансы спавна
- Обычный камень: 60%
- Медь: 25%
- Олово: 10%
- Железо: 5%

### Дропы ресурсов
- Обычный камень: 2 камня
- Медь: 1 камень + 1 медь
- Олово: 1 камень + 2 олово
- Железо: 1 камень + 1 железо
