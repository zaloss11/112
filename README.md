# Roblox Mining Game

3D игра по типу шахты в Roblox.

## Концепция
- Платформа 10x10 клеток (изначально доступна 1 клетка)
- На клетках спавнятся различные типы камней
- Игрок может добывать камни и продавать их торговцу
- Дополнительные клетки покупаются за монеты

## Типы камней
- Обычный камень: 2 камня
- Медь: 1 камень + 1 медь
- Олово: 1 камень + 2 олово
- И другие редкие ресурсы

## Структура проекта
- `ServerScripts/` - серверные скрипты
- `LocalScripts/` - клиентские скрипты
- `ReplicatedStorage/` - общие ресурсы
- `StarterGui/` - интерфейс
