# Шахта - Mining Game для Roblox

Трёхмерная игра в стиле шахты для платформы Roblox, где игроки добывают ресурсы, торгуют с NPC и расширяют свою шахту.

## 🎮 Особенности игры

### Основной геймплей
- **Платформа 10x10**: Игровое поле из 100 клеток (изначально доступна только одна)
- **Добыча ресурсов**: Кликайте по камням для их добычи
- **Типы камней**:
  - 🪨 **Обычный камень** - дает 2 камня (60% шанс спавна)
  - 🟫 **Медь** - дает 1 камень + 1 медь (25% шанс спавна)
  - ⬜ **Олово** - дает 1 камень + 2 олово (15% шанс спавна)

### Экономическая система
- **Торговец**: Продавайте ресурсы за монеты
- **Цены на ресурсы**:
  - Камень: 1₽
  - Медь: 5₽
  - Олово: 8₽
- **Расширение шахты**: Покупайте новые клетки за монеты (цена растет экспоненциально)

### 3D Окружение
- Полноценное трёхмерное игровое пространство
- Настроенное освещение и атмосфера
- Декоративный ландшафт с деревьями и холмами
- Интуитивно понятный интерфейс

## 🛠 Установка в Roblox Studio

### Метод 1: Ручное копирование
1. Откройте Roblox Studio и создайте новое место
2. В Explorer найдите **ServerScriptService**
3. Создайте новый **Script** и скопируйте содержимое `GameManager.lua`
4. Повторите для остальных серверных скриптов
5. В **StarterPlayer** → **StarterPlayerScripts** создайте **LocalScript** и скопируйте `ClientGUI.lua`

### Метод 2: Импорт файлов
1. Сохраните каждый .lua файл локально
2. В Roblox Studio используйте функцию импорта скриптов
3. Поместите файлы в соответствующие папки согласно структуре

## 🎯 Игровая механика

### Начало игры
1. Игрок появляется на спавне
2. Доступна одна клетка шахты (1,1)
3. На клетках периодически появляются камни (каждые 5 секунд, 50% шанс)

### Добыча ресурсов
1. Кликните по камню для его добычи
2. Получите ресурсы согласно типу камня
3. Камень исчезает, клетка освобождается для нового спавна

### Торговля
1. Подойдите к торговцу (используйте кнопку "К ТОРГОВЦУ" в GUI)
2. Кликните по прилавку для открытия торгового интерфейса
3. Продавайте ресурсы за монеты

### Расширение шахты
1. Накопите достаточно денег
2. В торговом интерфейсе купите новую клетку
3. Клетки разблокируются по спирали от центра
4. Стоимость каждой новой клетки увеличивается в 1.5 раза

## 🎨 Визуальные элементы

### Камни
- **Серый камень**: Обычный камень
- **Медный камень**: Оранжево-коричневый цвет
- **Оловянный камень**: Светло-серый металлический цвет

### Клетки
- **Разблокированные**: Светло-серый мрамор
- **Заблокированные**: Темно-серый бетон (нельзя пройти)

### GUI
- **Панель ресурсов**: Показывает количество ресурсов и денег
- **Панель шахты**: Информация о разблокированных клетках
- **Панель управления**: Быстрые переходы между локациями
- **Подсказки**: Обучающие советы для новых игроков

## ⚙️ Настройки игры

### Конфигурация в GameManager.lua
```lua
GameManager.Config = {
    GRID_SIZE = 10,           -- Размер сетки (10x10)
    TILE_SIZE = 8,            -- Размер клетки в студах
    SPAWN_INTERVAL = 5,       -- Интервал спавна камней (секунды)
}
```

### Настройка типов камней
```lua
GameManager.StoneTypes = {
    Stone = {
        name = "Камень",
        color = Color3.new(0.5, 0.5, 0.5),
        rarity = 0.6,  -- 60% шанс
        drops = {{item = "Stone", amount = 2}}
    },
    -- ... другие типы
}
```

### Экономические настройки
```lua
TraderManager.ResourcePrices = {
    Stone = 1,    -- 1₽ за камень
    Copper = 5,   -- 5₽ за медь
    Tin = 8,      -- 8₽ за олово
}
```
